# AWS IAM Audit Tool

Sample AWS security project demonstrating how to audit IAM users, MFA status, and access key hygiene.

## Problem

Cloud environments require regular IAM reviews to reduce security risk, improve compliance, and detect weak access controls.

## Solution

This project demonstrates a simple IAM audit workflow to identify risky accounts, missing MFA, and stale access keys.

## Checks Included

- IAM Users without MFA enabled
- Access keys older than threshold
- Basic IAM security review
- JSON output reporting

## Skills Demonstrated

- AWS IAM
- Cloud Security
- Compliance & Audit Thinking
- Python Automation
- Identity & Access Management
- Security Operations

## Example Use Case

Audit IAM accounts to identify security gaps and improve access governance in AWS environments.
